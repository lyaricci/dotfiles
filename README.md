# Lya's dotfiles

## Steps

### Git clone the repository

```bash
git clone https://github.com/lyaricci/dotfiles.git
```

### Run the following scripts to start setting up

```bash
install.sh
```

### Install and config ZSH

```bash
install-zsh.sh
```

### Install and config Neovim

```bash
install-neovim.sh
```

### Install and config Kitty terminal

```bash
install-kitty.sh
```
